# Cadena de Frío — Temperaturas y Recomendaciones

## Temperaturas estándar
| Producto | Temperatura | Notas |
|----------|-------------|-------|
| Flores   | 1°C - 3°C   | Evitar etileno. Control de humedad. |
| Pescado congelado | -18°C | Sin descongelación parcial. Max 72h. |
| Mariscos | -18°C | Muy sensible. Sin roturas de cadena. |
| Frutas tropicales | 8°C - 10°C | No mezclar con productos de etileno. |

## Alertas de temperatura
Si la temperatura se sale del rango ±2°C, el envío debe ser marcado para revisión inmediata.

## Empaque
- Flores: cajas ventiladas, hielo gel en verano.
- Pescado: cajas de poliestireno con hielo seco (CO2) para mantener -18°C.
- Frutas: cajas de cartón con separadores.
